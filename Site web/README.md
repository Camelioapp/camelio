# Camelio - Site web

Site Internet React/Vite pour la landing page de Camelio.

## Installation

Dans le dossier du projet :

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement Netlify

Build command :

```bash
npm run build
```

Publish directory :

```bash
dist
```

## Note

Les boutons d'appel à l'action pointent actuellement vers :

```txt
https://camelio.app
```

Tu peux modifier cette URL dans `src/LandingPage.jsx` si tu veux rediriger vers l'application, par exemple `https://app.camelio.app`.
