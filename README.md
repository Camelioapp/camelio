# Camelio, section d'accueil

Ce ZIP contient une version autonome de la première page Camelio.

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir l'URL affichée par Vite, généralement `http://localhost:5173/`.

## Fichier principal à modifier

```text
src/components/WelcomeScreen.jsx
```

## Build pour Netlify

```bash
npm run build
```

Le dossier généré sera `dist/`.
